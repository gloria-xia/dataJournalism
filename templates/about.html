Data Journalism- Gloria Xia

This project explores the number of hate crimes that were recorded by the NYPD in NYC from 2019-2023. I was interested in this topic after seeing the news around the rise in hate crimes during the COVID-19 pandemic. Hate crimes are offenses based on a person's identity. Given all of the news around this topic, I thought that it would be interesting to investigate the actual number of hate crimes throughout recent years.

Data pipeline:
  • 1. Download data from NYC Open Data as an CSV file
  • 2. Import the CSV file into Google Sheets and delete all of the unnecessary columns
  • 3. Convert the cleaned data into a json file
